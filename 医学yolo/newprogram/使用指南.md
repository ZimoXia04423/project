# 骨折分级智能检测系统 - 使用指南

## 项目概述

这是一个基于深度学习的骨折检测系统，采用四层架构设计（数据层、算法层、服务层、表现层），支持 YOLOv8 和 YOLOv10 模型，可自动判定骨折等级（轻度/中度/重度/未见骨折）。

## 系统架构

```
newprogram/
├── datasets/              # 数据层 - 数据集管理
├── preprocess/           # 算法层 - 图像预处理与数据增强
├── models/               # 算法层 - 模型配置与定义
├── train/                # 训练入口 - 模型训练
├── infer/                # 推理层 - 批量检测
├── service/              # 服务层 - 模型管理、图像处理、结果分析
├── gui/                  # 表现层 - 图形界面
├── main.py               # 程序入口
├── LoginWindow.py        # 登录窗口
└── UiMain.py            # 主界面
```

### 6. 批量推理

```python

    image_path='test.jpg',
```

## 核心功能说明

### 图像预处理 (4 步流程)

1. **灰度归一化**: 将像素值映射到 [0, 255] 范围
2. **CLAHE 增强**: 提升骨皮质边界清晰度
3. **噪声抑制**: 中值滤波去除随机噪声
4. **尺寸标准化**: 缩放至 640×640

### 数据增强 (5 种策略)

1. **随机翻转**: 水平翻转 (50% 概率), 垂直翻转 (20% 概率)
2. **轻微旋转**: -15° 到 +15° 随机旋转
3. **亮度对比度**: 80%-120% 随机调整
4. **缩放平移**: 90%-110% 缩放，±10% 平移
5. **Mosaic/MixUp**: 可选增强（默认关闭）

### 骨折分级规则

| 等级 | 条件 | 描述 |
|------|------|------|
| 重度骨折 | fracture + periostealreaction | 检测到骨折伴骨膜反应 |
| 中度骨折 | fracture + (softtissue 或 pronatorsign) | 检测到骨折伴软组织肿胀/旋前征 |
| 轻度骨折 | 仅 fracture | 检测到单纯骨折 |
| 未见骨折 | 无 fracture | 未检测到明确骨折征象 |

### 检测的 4 类病灶

- **fracture** (红色): 骨折
- **periostealreaction** (橙色): 骨膜反应
- **pronatorsign** (绿色): 旋前征
- **softtissue** (蓝色): 软组织肿胀

## 性能优化建议

1. **GPU 加速**: 使用 CUDA 可显著提升训练和推理速度
2. **批量处理**: 使用 `predict_batch` 进行批量推理
3. **模型导出**: 导出为 ONNX 格式可在 CPU 上获得更好性能
4. **图像预处理**: CLAHE 参数可根据实际图像质量调整

## 常见问题

### Q: 模型加载失败？
A: 检查权重文件路径是否正确，确保文件存在。

### Q: CUDA 内存不足？
A: 减小 `batch_size` 或使用更小的 `img_size`。

### Q: 检测精度不高？
A: 尝试增加训练轮数、调整学习率或使用更多训练数据。

### Q: 如何扩展新功能？
A: 系统采用模块化设计，可在对应层中添加新功能：
- 新模型：在 `models/` 中定义
- 新预处理：在 `preprocess/` 中添加
- 新界面：在 `gui/` 中扩展
