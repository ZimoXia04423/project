## 📊 效果图说明

### 预处理效果（4 步流程）

1. **灰度归一化**: 将像素值统一映射到 [0, 255] 范围
   - 解决不同设备采集的图像亮度差异问题
   
2. **CLAHE 增强**: 对比度受限自适应直方图均衡化
   - 提升骨皮质边界的清晰度
   - 参数：clip_limit=2.0, grid_size=8x8
   
3. **噪声抑制**: 中值滤波
   - 去除随机噪声，同时保持边缘
   - 强度：0.6（可调）
   
4. **尺寸标准化**: 缩放至 640×640
   - 统一模型输入尺寸

### 数据增强（5 种策略）

1. **随机翻转**: 
   - 水平翻转概率：50%
   - 垂直翻转概率：20%（医学图像谨慎使用）

2. **轻微旋转**: 
   - 范围：-15° 到 +15°
   - 增强对体位变化的鲁棒性

3. **亮度对比度扰动**:
   - 亮度：80%-120%
   - 对比度：80%-120%
   - 增强对曝光差异的适应性

4. **随机缩放平移**:
   - 缩放：90%-110%
   - 平移：±10%
   - 提升对位置变化的泛化能力

5. **Mosaic/MixUp**（可选）:
   - 默认关闭
   - 需谨慎使用，避免破坏医学结构

## 🔧 自定义参数

您可以在代码中调整参数以获得不同效果：

### 调整预处理参数
```python
from preprocess import ImagePreprocessor

preprocessor = ImagePreprocessor(
    target_size=(640, 640),    # 目标尺寸
    clip_limit=3.0,            # CLAHE 对比度限制（1.0-5.0）
    grid_size=(16, 16),        # CLAHE 网格大小
    use_denoise=True,          # 是否去噪
    denoise_strength=0.8       # 去噪强度（0-1）
)
```

### 调整数据增强参数
```python
from preprocess import DataAugmenter

augmenter = DataAugmenter(
    h_flip_prob=0.7,           # 水平翻转概率
    v_flip_prob=0.1,           # 垂直翻转概率
    rotation_range=(-20, 20),  # 旋转角度范围
    brightness_range=(0.7, 1.4),  # 亮度范围
    contrast_range=(0.7, 1.4),    # 对比度范围
    scale_range=(0.85, 1.15),     # 缩放范围
    translate_range=(-0.15, 0.15) # 平移范围
)
```

## 📝 注意事项

1. **医学图像特殊性**:
   - 垂直翻转可能改变器官位置，需谨慎使用
   - 增强不应破坏解剖结构语义

2. **参数选择**:
   - 根据实际图像质量调整 CLAHE 参数
   - 噪声过强时增加去噪强度

3. **效果验证**:
   - 预处理后应能更清晰看到骨皮质边界
   - 增强后的图像应保持医学合理性
